## Class Presentation Topics

You can pick one or more presentation topics and present them to the class for extra credit. 
This is completely voluntary, but it will benefit the whole class.

### **Potential Student Presentation Topics**

***Note**: Each bullet item is a topic, they are just grouped by area.*

#### **Software Development Practices**

* Evolution of **Agile vs. Waterfall vs. DevOps** methodologies
* **Test-Driven Development (TDD):** pros, cons, and best practices
* **Continuous Integration / Continuous Deployment (CI/CD)** pipelines with GitHub Actions, Jenkins, or GitLab CI
* **Code Review Best Practices** and their impact on software quality
* **Pair Programming & Mob Programming:** effectiveness and real-world usage

#### **Programming Languages & Tools**

* Why **Java is still relevant** in 2025 (compared to Python, Go, Rust, etc.)
* An introduction to **Rust** and its approach to memory safety
* **Kotlin vs. Java** for backend and mobile development
* **Git branching strategies** (Git Flow, trunk-based development, etc.)
* **IDEs vs. lightweight editors**: how tool choice shapes developer productivity

#### **Software Architecture & Design**

* **Microservices vs. Monoliths:** tradeoffs in real systems
* **SOLID principles** explained with real-world examples
* **MVC and its modern alternatives** (MVVM, MVP, etc.)
* **Design Patterns in Practice**: Singleton, Factory, Observer, etc.
* **Event-Driven Architectures** in cloud applications

#### **Emerging Tech in Software Development**

* **Agentic AI in software engineering** (AI pair programmers, copilots, autonomous agents)
* **Quantum Computing**: what developers should know today
* **WebAssembly (WASM):** changing how we deploy applications
* **Serverless computing**: when and why to use it
* **Low-code / No-code platforms**: impact on professional developers

#### **Security & Ethics**

* **OWASP Top 10** vulnerabilities in web applications
* **Secure coding practices** every developer should know
* **Open Source Software Licenses** (MIT, GPL, Apache) and why they matter
* **Ethical implications of AI in software systems**
* **Supply Chain Attacks** and how to protect against them

#### **Career & Professional Development**

* **How to contribute to open source projects**
* **Technical interviewing strategies** for software engineering roles
* **Building a developer portfolio on GitHub**
* **Certifications in software development** (AWS, Microsoft, Google)
* **Soft skills for software engineers**: communication, teamwork, and leadership
